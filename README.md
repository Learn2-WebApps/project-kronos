# 프로젝트 KRONOS — HRD용 AI 기반 추리 게임 기획

**학습자가 AI 용의자들과 채팅하며 거짓말을 찾아내는 HRD 추리 게임. AI 리터러시·비판적 사고·교차검증 능력 학습 목적.**

## 📂 폴더 구조

```text
project-kronos/
├── README.md
├── CHANGELOG.md

│   ├── 00-MASTER.md

```

### 폴더별 역할
- **docs/**: 마스터 시나리오 및 주요 설계 문서
- **docs/characters/**: 용의자 및 참고인 5인의 상세 배경과 알리바이
- **docs/prompts/**: LLM 구동을 위한 시스템 및 캐릭터 페르소나 프롬프트
- **docs/learner/**: 학습자용 브리핑 시트 및 활동 템플릿
- **docs/facilitator/**: 퍼실리테이터 운영 가이드 및 개입 포인트
- **src/**: 프로토타입 또는 시연용 소스 코드
