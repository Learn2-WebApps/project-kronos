## 4. 시간 충돌 점검 결과

### 4.1 같은 시각의 여러 사건 — 모순 없음
- 21:00 시점에 정민호는 한지훈과 헤어진 직후 → 오세라 회동으로 이동. 동선 자연스러움.
- 21:00 시점에 윤서경은 부산 호텔 객실에 있음. 정민호와 무관.
- 23:47 시점에 정민호는 카페에서 메일 발송. 오세라는 자택 (본인 주장).

### 4.2 정민호의 거짓 진술 시간선 일관성 점검
- L-01 ("11시까지 식사") + L-07 ("식사 후 곧장 귀가") → 두 거짓이 결합하면 23:47 메일 발송 시점과 충돌. 학습자가 이 충돌을 감지하면 KEY-1b 도달.
- L-01만 깨지면 → 21:00 이후 행적 공백 노출 → 자연스럽게 KEY-1b로 연결.

### 4.3 오세라의 검증 불가 알리바이
- T-06 ("자택에서 혼자") → 일관됨. 의도적으로 반증 불가.
- F-23 (회동)은 정민호 자백 시에만 노출. 오세라는 끝까지 부인.

### 4.4 윤서경 함정 알리바이
- F-16 (입증자 없음) + F-17 (비서의 "방해 금지" 진술) 결합 시 의심 해소.
- 학습자가 F-17까지 도달하지 못하면 윤서경 의심 유지. 의도된 함정.

## 6. 운영 시 주의 사항

1. **F-23 (회동)은 정민호와 오세라만 아는 정보**. 다른 캐릭터(특히 강혜린)가 이 정보를 알고 있지 않도록 페르소나에 명시.
2. **회동 장소 "삼성동 호텔 라운지"는 내부 고정값**. 학습자에게는 정민호 자백 시 "서울 모처"로만 답하도록.
3. **23:47 메일 발송 시각은 자동 노출** (F-01). 학습자가 모르고 시작하면 안 됨.
4. **F-24 (자택 도착 23:30 추정)는 ECHO 추론**. 캐릭터가 직접 발화하지 않음. ECHO가 시간선 정리 시 환기.
5. **윤서경 부산 출장(F-15)은 자동 노출**, 알리바이 공백(F-16)은 추궁 시 노출, 비서 진술(F-17)은 강혜린에게 명시 요청 시 노출. 3단 구조 유지.
6. **10/15 ~ 10/20 사이 인사위원회 절차**는 강혜린 통해 시간 단위로 확인 가능하나, 사건 본질과는 무관. 학습자가 깊이 파지 않아도 됨.
